TagDialog

Get Extension

これは何か

Thunderbird のタグ追加/編集を簡単する拡張機能

キーボードをメインに使う人でも使えるようになっています

機能

メッセージ -> Tag Dialog または iキーでダイアログが表示される
付けたいタグをカンマ(,)(デフォルトでは)区切りで入力することで複数のタグを付けられる

入力途中と思われるタグには候補が上がり、以下の操作が行える

  • Ctrl + n, Ctrl + RIGHT, Ctrl + DOWNで次の候補
  • Ctrl + p, Ctrl + LEFT, Ctrl + UPで前の候補
  • Ctrl + ENTERで確定

また、チェックボックスをオン/オフにすることでも既存タグの追加/削除が行える

コンテキスト(右クリック)メニュー

タグのチェックボックスを右クリックすることで専用のメニューが出てくる

  • 編集(edit)はそのタグの名前や色を変更できる
  • コピー(copy)はタグ名をクリップボードにコピー
  • 削除(delete)はタグの削除
  • フィルタ(filter)メニューからはフィルターの編集を行える
  • オプション(option)はTagDialogのオプション設定ができる(以下のオプションの項を参照)

オプション

  • タグの区切り文字（デフォルト:カンマ(,)）
  • 新規タグ追加時の確認をするかどうか（デフォルト:オン）
  • 新規タグ追加時のデフォルトの色

ビルド方法

        $ git clone git://github.com/teramako/TagDialog.git 
        $ cd TagDialog
        $ sh build.sh
