Vimperator

Vimperator とは

  • キーボードによるブラウジングを提供する
    • エディタの Vim のような操作
      • j,k で上下スクロール、ggで先頭へ、Gで末尾へ etc...
    • Hit-a-Hint の様なリンク選択
  • UI にまで手を加えている
    • ロケーションバーが消えたり
    • ステータスバーが Vim っぽくなったり
  • コマンドライン機能 & 補完機能
  • 優れたカスタマイズ性
    • (UNIXライクな) RC ファイルによる設定
    • プラグイン機能

Vimperator周りの系統図

多くの拡張やソフトが Vimperator の影響を受けている

  • UNIX系ツールからFirefox拡張機能へ
    • まずmozless
    • Emacsライクな操作を可能にするFiremacs
    • キーボードからリンク選択を可能にするHit-a-Hint (Firefox 2.0まで) → 現在 LoL へフォーク
    • Vim的カスタマイズ性を備えた Vimperator (超オススメ)
    • よりEmacs的に拡張可能な KeySnail (オススメ)
  • Firefox拡張機能からGoogleChrome拡張機能へ
    • Vimlike-Smooziee
    • Vimium
    • Vrome
  • OperaではVimperator for Opera なんてのが
  • 最近では、luakit(WebKit + Lua) なんてのも

入門者たちへ

  1. 導入直後、十分に驚きを味わってください。
  2. 導入して再起動後に表示されるヘルプを良く読みましょう
    • メニューバーの表示方法、Vimperatorの無効化方法が書かれています
    • チュートリアルもあります
  3. Vimperator日本語ヘルプ拡張があるで、英語が苦手なら導入すると良いかも

中毒者たちの症状

  1. ブログに設定した RC ファイルやプラグインを晒す
  2. Vim を使い始める
  3. 「Vimperatorのせいで他のブラウザに乗り換えられない」などとTwitterに呟く
  4. 「自分がFirefoxを使っているのはVimperatorがあるから」などとTwitterに書き込む
  5. 「Vimperatorはブラウザです！」などと突然Twitterに書き込む
  6. 最新プラグイン入手のために、Subversion, Git を使い始める
  7. プラグインを作ってみようとする
  8. 最新版入手のためにMercurialを導入し、ビルドし始める
  9. メーリングリストを購読し始める
